# Java&MFC版badApple

由于近段时间有人私信我代码，而原来百度云共享的内容也失效了，我就把代码传到这里来了

## 代码说明

将2.rar解压到D盘SL目录（如果没有请创建）下，运行exe文件即可
java文件处理逐帧截取的视频图片，将图片转换为数字，具体处理可参看java代码
MFC为VS2012创建，它读取D盘SL\2目录下的txt文件生成动画

本项目为2014年初学Java时编写，诸多问题，敬请见谅
